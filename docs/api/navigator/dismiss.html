Navigator.dismiss

dismiss([payload])

Arguments

  1. payload (Object): An optional payload to pass back to the presenting screen. This is what the promise returned from present(...) will resolve with.

Example Usage

import Navigator from 'native-navigation';

Navigator.dismiss();
